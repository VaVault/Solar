# Solar

A simple website that visualizes the solar system in 3D using [Three.js](https://threejs.org/).

## Usage

Open `index.html` in a modern web browser. Use the mouse to rotate and zoom around the scene.

To host the page with GitHub Pages, push the repository to GitHub and enable Pages in the project settings. The site will then be available at `https://<username>.github.io/<repository>/`.
